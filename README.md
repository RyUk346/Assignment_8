# SkillSphere – Online Learning Platform

SkillSphere is a modern, responsive online learning platform built with Next.js. It allows users to explore courses, view instructors, manage profiles, and access protected course details with authentication.

---

## Live URL

https://assignment-8-ochre.vercel.app/

---

## Project Purpose

The purpose of this project is to create a user-friendly learning platform where users can:

- Browse courses
- View course details
- Authenticate (login/register)
- Manage personal profiles
- Experience smooth UI with modern animations

---

## Key Features

- Authentication system (Email/Password + Google login)
- User profile management with update functionality
- Protected routes (course details & profile pages)
- Course listing with search functionality
- Popular & trending course sections
- Learning tips section with animations
- Fully responsive UI (mobile, tablet, desktop)
- Toast notifications for user actions
- Smooth animations using Motion
- Mobile-friendly navbar with dropdown menu
